## About Fleet Tracker

## Setup

Use will need `node.js` and `pnpm` to setup the project:

```shell
npm install -g pnpm
```

Next, copy `.env.example` to `.env` and change `NEXT_PUBLIC_API_URL` if needed.

## Usage

Run the following command to start the development server:

```shell
pnpm dev
```
